# Class: CompactSign

The CompactSign class is a utility for creating Compact JWS strings.

**`example`** ESM import
```js
import { CompactSign } from 'jose/jws/compact/sign'
```

**`example`** CJS import
```js
const { CompactSign } = require('jose/jws/compact/sign')
```

**`example`** Deno import
```js
import { CompactSign } from 'https://deno.land/x/jose@v3.20.4/jws/compact/sign.ts'
```

**`example`** Usage
```js
const encoder = new TextEncoder()

const jws = await new CompactSign(encoder.encode('It’s a dangerous business, Frodo, going out your door.'))
  .setProtectedHeader({ alg: 'ES256' })
  .sign(privateKey)

console.log(jws)
```
